# Автоматизированный агрегатор новостей

Консольное приложение на Spring Boot для автоматического сбора, обработки и анализа новостей из RSS-лент и HTML-страниц. 
Проект демонстрирует применение принципов SOLID, GRASP, инверсии зависимостей и юнит-тестирования.

## Технологии

- **Java 17+**
- **Spring Boot 3** (pring Data JPA
- **База данных**: H2 
- **Jsoup** – парсинг RSS/HTML
- **Mockito + JUnit 5** – юнит-тесты
- **Maven** – сборка

## Функциональные возможности

-  Автоматический сбор новостей из RSS-фидов (Lenta.ru, Habr, RIA, TASS, Коммерсантъ и др.)
-  Очистка текста от рекламы и лишних тегов
-  Определение категории (политика, экономика, спорт, наука, технологии, культура, происшествия, общество)
-  Извлечение ключевых слов и краткого резюме (summary)
-  Поиск и фильтрация по ключевым словам, категории, источнику, дате
-  Сортировка по дате, популярности, источнику
-  Экспорт в CSV, JSON, HTML
-  Аналитика: статистика по категориям, популярные темы (тренды)
-  Ручное и автоматическое обновление (опционально)
-  Консольный интерфейс с командами
